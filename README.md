# user-manager
User manager is user service for managing users, groups, credentials, user and domain blacklists for Containerum

## Prerequisites
MongoDB 


## Installation

### Manually


### Using Helm

```
  helm repo add containerum https://charts.containerum.io
  helm repo update
  helm install containerum/user-manager
```
